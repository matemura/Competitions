Problem 1: Pareto's Domination (2) [AUPC 2008 Problem E, 2010]

FJ wants to improve the management of his farm and has enrolled in
a microeconomics class at the Manure Institute of Technology.

In one of his classes, he learned about 'Pareto domination' (which
is apparently important for game theory). Pareto-domination has to
do with a relation between two M-dimensional points x_i = (x_i1,
x_i2, ..., xi_M) and x_j = (y_j1, y_j2, ..., y_jM) where (2 <= M
<= 3; 0 <= x_ik <= 1,000,000,000).

FJ recalls that a point x_i Pareto-dominates a point x_j if, for
each k (1 <= k <= M), the inequality x_ik >= x_jk holds.

FJ must complete a problem set on Pareto-domination. For each
problem, he is given a set of N (1 <= N <= 50,000) distinct points.
He must identify all points that are not Pareto-dominated by any
other points in the set.

Help FJ check his answer by writing a program to output the number
of points that are not Pareto-dominated by any other point.

TIME LIMIT: 1.5 seconds

MEMORY LIMIT: 64MB

PROBLEM NAME: dom2

INPUT FORMAT:

* Line 1: Two space separated integers: N and M

* Lines 2..N+1: Line i contains M space-separated integers: point x_i

SAMPLE INPUT (file dom2.in):

4 2
0 0
1 1
2 0
0 2

INPUT DETAILS:

There are 4 points in 2D: (0,0), (1,1), (2,0), (0,2).

OUTPUT FORMAT:

* Line 1: A single integer: the number of points that are not
        Pareto-dominated by any other point

SAMPLE OUTPUT (file dom2.out):

3

OUTPUT DETAILS:

Only the point (0,0) is dominated.

Problem 2: Moo University -- Financial Aid (2) [Hal Burch, 2004]

Bessie noted that although humans have many universities they can attend,
cows have none. To remedy this problem, she and her fellow cows formed a
new university called The University of Wisconsin-Farmside, "Moo U" for short.

Not wishing to admit dumber-than-average cows, the founders created an
incredibly precise admission exam called the Cow Scholastic Aptitude Test
(CSAT) that yields scores in the range 1..2,000,000,000.

Moo U is very expensive to attend; not all calves can afford it. In fact,
most calves need some sort of financial aid (0 <= aid <= 100,000). The
government does not provide scholarships to calves, so all the money must
come from the university's limited fund (whose total money is F, 0 <= F <=
2,000,000,000).

Worse still, Moo U only has classrooms for an odd number N (1 <= N <=
19,999) of the C (N <= C <= 70,000) calves who have applied. Bessie wants
to admit exactly N calves in order to maximize educational opportunity. She
still wants the median CSAT score of the admitted calves to be as high as
possible.

Recall that the median of a set of integers whose size is odd is the middle
value when they are sorted. For example, the median of the set {3, 8, 9, 7,
5} is 7, as there are exactly two values above 7 and exactly two values
below it.

Given the score and required financial aid for each calf that applies, the
total number of calves to accept, and the total amount of money Bessie has
for financial aid, determine the maximum median score Bessie can obtain by
carefully admitting an optimal set of calves.

PROBLEM NAME: finance2

INPUT FORMAT:

* Line 1: Three space-separated integers: N, C, and F.

* Lines 2..C+1: Two space-separated integers per line: The first is
        the calf's CSAT score; the second integer is the required
        amount of financial aid the calf needs.

SAMPLE INPUT (file finance2.in):

3 5 70
30 25
50 21
20 20
5 18
35 30

OUTPUT FORMAT:

* Line 1: A single integer, the maximum median score that Bessie can
        achieve. If there is insufficient money to admit N calves,
        output -1.

SAMPLE OUTPUT (file finance2.out):

35

OUTPUT DETAILS:

If Bessie accepts the calves with CSAT scores of 5, 35, and 50, the median is
35. The total financial aid required is 18 + 30 + 21 = 69 <= 70.

Problem 3: Palindrome (4) [Richard Peng, 2007]

The entire herd of N (1 <= N <= 100,000) cows has become fascinated
with palindromes (a palindrome is a string that reads the same both
forwards and backwards).

Each cow chooses a lower-case letter ('a'..'z') and paints it on
her forehead before joining the growing line of cows. The cows then
ask Farmer John to determine the length of the longest odd-length
palindrome centered on each cow.

Help FJ confirm his count is correct.

PROBLEM NAME: palin4

INPUT FORMAT:

* Line 1: A single integer: N

* Lines 2..N+1: Line i + 1 contains the single character painted on
        the forehead of cow i.

SAMPLE INPUT (file palin4.in):

6
a
a
a
b
a
a

OUTPUT FORMAT:

* Lines 1..N: Line i contains the length of the longest palindrome
        with its center character at i.

SAMPLE OUTPUT (file palin4.out):

1
3
1
5
1
1
