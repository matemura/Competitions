Problem title: red
Problem number: 2
Round: Round #6 - contest
Solution: Theorem

The number of possibilities to color k of n points placed in a circle, so that no two consecutive points be colored is

n/(n-k)*Comb(n-k,k)

where by Comb(n-k,k) we understand combinations of n-k taken in k.

Demonstration

Let's number the points from 1 to n.

Let's calculate in how many ways we can color the k points (so that no two consecutive points be red) considering that point 1 is not red.

We place the n-k non-colored points on a circle, one of them being point 1. Between these points we insert the k points colored in red (this can be done in Comb(n-k,k) ways).

Now let's calculate in how many ways we can color the k points, considering that point 1 is red.

We select the select the n-k points of the circle, amongst which we have to insert k-1 red points, considering that we can't place red points next to point 1. Therefore there are Comb(n-k-1.k-1) ways).

Total:

Comb(n-k-1.k-1)+Comb(n-k,k)=n/(n-k)Comb(n-k,k).

Implementation:

We calculate the combinations using the recurrence relationship:

Comb(n,k)=Comb(n-1.k)+Comb(n-1.k-1)

Comb(n,n)=Comb(n,0)=1;

Large number addition implementation is necessary.
